# Skill-Track Backend

The API and database management system for **Skill-Track**, responsible for handling user authentication, skill verification, and achievement tracking.

## Backend Tech Stack
- **Runtime:** Node.js (Express)
- **Database:** MongoDB
- **Authentication:** JWT, Firebase
- **Data Management:** Mongoose, RESTful API principles

## Frontend Connection
This backend serves the Skill-Track frontend application. You can find the frontend repository here:
- [Skill-Track Frontend Repository](ADD_FRONTEND_REPO_LINK_HERE)

## Getting Started
1. Clone the repository: `git clone [YOUR_BACKEND_REPO_URL]`
2. Install dependencies: `npm install`
3. Create a `.env` file and add your MongoDB and Firebase configuration.
4. Run the server: `npm run start`
